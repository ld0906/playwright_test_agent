EP 08/CI 与边界

PR Workflow · playwright.yml

YAML .github/workflows/playwright.yml 每个 PR / push 都跑 
name: Playwright Tests
on: { pull_request: { branches: [main] } }

jobs:
  test:
    runs-on: ubuntu-latest
    timeout-minutes: 15
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4 // node 20
      - run: npm ci
      - run: npx playwright install --with-deps chromium

      - name: Start demo-app
        run: |
          cd ../demo-app && npm run dev &
          npx wait-on http://localhost:5173 -t 30000

      - run: npx playwright test --reporter=list,html

      - uses: actions/upload-artifact@v4
        if: failure()
        # traces + screenshots，调试用
DETAIL · 01
--with-deps 装系统依赖，漏了 Ubuntu 会缺库报错
DETAIL · 02
用 wait-on 等服务起来，不要 sleep
DETAIL · 03
timeout 15 分钟卡死。超时是真问题，不允许"机器慢"作借口
