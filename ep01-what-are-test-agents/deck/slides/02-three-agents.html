EP 01 / 课程地图

先把名词搞清楚

AGENT · 01
Planner
会自己点页面 + 观察 + 写笔记的初级测试
Input seed.md + 运行中的 app
Output Markdown 测试计划
AGENT · 02
Generator
把计划翻译成可运行 spec 的初级测开
Input 测试计划 + 项目结构
Output Playwright TypeScript spec
AGENT · 03
Healer
UI 改了帮你修脚本的初级运维
Input 失败的 spec + 当前 DOM
Output 修复后的 spec
三个代理本质都是 prompt + MCP，没有黑魔法。关键不是 AI 多聪明，是你给它的输入好不好。
