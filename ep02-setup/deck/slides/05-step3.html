配置 Playwright MCP最关键一步
JSON .mcp.json 放在项目根 · 跟 demo-app / tests-agents 平级 
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["-y", "@playwright/mcp@latest"]
    }
  }
}
DETAIL · 01
用 npx，不需提前 npm install。第一次启动自动下载 @playwright/mcp，稍慢，后面就快了。
DETAIL · 02
Cursor 用户改 ~/.cursor/mcp.json，JSON 结构完全一样。
