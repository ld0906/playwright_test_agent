# 2. 一行启动
npm run dev
✓ 预期
  • 终端显示 Local: http://localhost:5173
  • 浏览器打开能看到登录页
  • 用 alice / password 登录，看到书籍列表
⚠ 常见踩坑
  • 端口 5173 被占：lsof -i:5173 找出来杀掉，或改端口
  • Node 版本太低：本课要求 node -v ≥ 20
